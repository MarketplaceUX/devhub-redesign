Firefox Marketplace Developer Specs

This site contains the design specifications for the evantual redesign of the Firefox Marketplace's Developer sections (aka the devhub.) This document is currently a living document as the design process is underway.

Get started by reading the Project Overview »

If you would like to contribute to the design, please file an issue against this document using the link in the top navigation or feel free to clone and make pull requests against the document at: the spec github repo. All the individual design artifacts that are generated to create this spec can be found in the master branch of this repo.
