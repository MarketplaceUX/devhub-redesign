Marketplace Config File Research

Preliminary recommendations for Marketplace Config File

Test #1

Date: July 25, 2014

This session tested the new MDN documentation page for Marketplace Config File.

  • Due to test instruction not linking to the correct HTML file, all 6 participants failed to submit a config file that the instruction had asked for.
  • However, 2 participants reported the error. When an error is reported, UserTesting allows rerun.
  • After correcting the test instruction, the 2 tests were reran and analysed.
  • Our sample size had always been 6. To get 4 more respondents, we ran Test #2.
